## ğŸš€ Getting Started

### For Users (Workers/Migrants)

#### Starting a Conversation
1. Navigate to the Agencies page (`/agencies`)
2. Browse available recruitment agencies
3. Click the **"Message Agency"** button on any agency card
4. You'll be automatically redirected to the messaging page with the conversation started

#### Sending Messages
1. Go to Messages page (`/messages`)
2. Select a conversation from the left sidebar
3. Type your message in the input field at the bottom
4. Press Enter or click the send button
5. Your message appears instantly on the right side

#### Reading Messages
1. New messages appear in real-time
2. Unread count shows on each conversation
3. Messages are automatically marked as read when you view them
4. Green dot (â—) indicates you're connected for real-time updates

### For Agencies

#### Receiving Messages
1. Users can message you from your agency profile
2. You'll receive a notification when a new message arrives
3. Access messages from the Messages page (`/messages`)

#### Replying to Users
1. Navigate to Messages page
2. Select the conversation with the user
3. Type and send your reply
4. Users receive messages in real-time

## ğŸ› ï¸ Troubleshooting

### Messages Not Sending
**Issue**: Click send but message doesn't appear
**Solutions**:
1. Check your internet connection
2. Look for the connection status (should be green â—)
3. Refresh the page and try again
4. Log out and log back in

### Not Receiving Real-time Updates
**Issue**: Messages don't appear without refreshing
**Solutions**:
1. Check connection status indicator
2. Ensure you're logged in
3. Try closing other tabs with the app open
4. Clear browser cache and reload

### Can't Find Conversations
**Issue**: Conversations list is empty
**Solutions**:
1. Start a new conversation by messaging an agency
2. Check if conversations are archived
3. Ensure you're using the correct account

### Message Button Not Showing
**Issue**: "Message Agency" button not visible
**Solutions**:
1. Make sure you're logged in
2. Agencies cannot message other agencies
3. Refresh the agency profile page
